## Упражнения

Формат для LMS: задания без кода, каждое имеет один из трёх типов — открытый ответ, выбор ответа или сопоставление. В ноутбуке `Тест_к_теории.ipynb` для каждого задания явно указаны тип, варианты ответа при необходимости и правильный ответ.

1. Последовательный ансамбль — сопоставление
2. Остатки в регрессии — открытый ответ
3. Learning rate — открытый ответ
4. Маленький learning rate — выбор ответа
5. Сложность базовых деревьев — выбор ответа
6. Early stopping — открытый ответ
7. Функция потерь — сопоставление
8. Масштабирование — выбор ответа
9. Внешние библиотеки бустинга * — выбор ответа
10. Сравнение с Random Forest — сопоставление
