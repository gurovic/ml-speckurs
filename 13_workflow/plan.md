# План занятия 25. Рабочий процесс машинного обучения

1. От задачи заказчика к измеримой ML-задаче
2. Объект, признаки и целевая переменная
3. Единица наблюдения и момент прогноза
4. Выбор метрики по цене ошибок
5. Разделение train, validation и test до анализа и настройки
6. Исследование только обучающих данных
7. Утечки данных: признаки из будущего и подглядывание в test
8. Baseline как обязательная точка отсчёта
9. Обучение преобразований только на train
10. Единая последовательность преобразований для новых объектов
11. Воспроизводимость: random_state, версии данных и экспериментов
12. Сравнение моделей по одинаковому протоколу
13. Анализ ошибок, а не только одной метрики
14. Финальное тестирование и запрет дальнейшей настройки по test
15. Внедрение, мониторинг и изменение данных со временем
16. Чек-лист ML-эксперимента

## Упражнения

1. Сформулировать объект, цель и момент прогноза
2. Выбрать метрику для прикладной задачи
3. Найти утечку в списке признаков
4. Разнести операции между train, validation и test
5. Построить baseline
6. Найти ошибку в масштабировании
7. Составить таблицу экспериментов
8. Предложить мониторинг модели после внедрения
