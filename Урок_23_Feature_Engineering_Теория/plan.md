## Упражнения

Формат для LMS: задания без кода, каждое имеет один из трёх типов — открытый ответ, выбор ответа или сопоставление. В ноутбуке `Тест_к_теории.ipynb` для каждого задания явно указаны тип, варианты ответа при необходимости и правильный ответ.

1. Безопасный признак — сопоставление
2. Нормировка на длительность — открытый ответ
3. Логарифм для длинного хвоста — выбор ответа
4. Признак-взаимодействие — открытый ответ
5. Признаки из даты — сопоставление
6. One-hot или ordinal — сопоставление
7. Пропуск как сигнал — выбор ответа
8. Неизвестная категория — выбор ответа
9. Агрегаты без утечки — выбор ответа
10. Проверка полезности признака — выбор ответа
