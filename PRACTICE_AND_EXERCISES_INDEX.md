## Дополнительные материалы для практик в формате leaderboard

Для занятий 26 и 28 есть отдельные папки с данными, baseline, авторским решением и учительской проверкой внутри соответствующих тем:

- `Урок_26_Линейная_регрессия_Практика/competition/`
- `Урок_28_Логистическая_регрессия_Практика/competition/`

Итоговый проект по модулю вынесен из папок уроков в отдельную корневую папку и рассчитан на четыре занятия по 90 минут: постановка задачи, подготовка данных, сравнение моделей, финальное решение и защита. Сабмиты проверяет StackMoreLayers:

- `final_ensemble_competition/`

Внутри каждой папки:

- `data/train.csv`, `data/test.csv`, `data/sample_submission.csv`;
- `data/private_target.csv` — служебный файл, не выдавать ученикам;
- `simple_baseline.ipynb` — можно выдать ученикам по решению учителя;
- `author_solution.ipynb` — авторское решение;
- `solution_notes.md` — разбор идей;
- `score_submissions.ipynb` — учительская проверка файлов команд.

В итоговом проекте `score_submissions.ipynb` используется только для служебной проверки комплекта автором курса; официальный результат и таблицу лидеров формирует StackMoreLayers. Шаблон отчёта для защиты находится в `final_ensemble_competition/project_report_template.md`.
